LIVE Page Design
Single-card "Caddie Whisper" design. One piece of info at a time, full focus.
Layout:

Header: Red pulsing dot + "LIVE" text + current time (right aligned)
Main: Single card, centered, max-width 600px
Footer: Pagination dots + prev/pause/next controls

Card Types to support:

Hole Info (course context)

Badge: "On Course"
Hole number (large, serif font)
Par + yards
Quote (serif italic) — course history or caddie tip
Tip line below


Weather

Badge: "Weather"
Temperature (large)
Conditions + wind


Time Countdown

Badge: "Coming Up"
Countdown (e.g., "2h 14m") in accent color
Event name + details


Logistics

Badge: "Logistics"
Status icon (checkmark for confirmed)
Status text + detail



Behavior:

Cards auto-rotate every 8 seconds
Clicking dots jumps to that card
Prev/Next buttons navigate manually
Pause button stops auto-advance
Keyboard: Left/Right arrows, Spacebar to pause

Animation:

Card exit: fade out + slide up 20px (400ms)
Card enter: fade in + slide up from 20px below (500ms)
Easing: cubic-bezier(0.22, 1, 0.36, 1)

Typography:

Hole numbers and temperatures: Serif font (Cormorant Garamond), large
Quotes: Serif italic
Everything else: Sans-serif (Inter)

Colors:

Live red: #B8372D
Accent (countdown): #C9A227
Card background: white with subtle shadow
Card border: rgba(26, 47, 35, 0.06)
