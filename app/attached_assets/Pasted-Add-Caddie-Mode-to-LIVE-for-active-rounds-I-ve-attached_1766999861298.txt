Add "Caddie Mode" to LIVE for active rounds. I've attached a reference file (caddie-mode-reference.html) — match this exactly.
When to Trigger
When round.status === 'in_progress' for the current user's group, LIVE automatically switches to Caddie Mode.
Layout (top to bottom)
1. Sticky Course Header

"LIVE — Round in Progress" with red pulsing dot
Course name (large, serif font)
Weather: temp, conditions, wind
Round progress: "Hole X of 18"

2. Hole Progress Dots

18 numbered dots
Completed holes = green filled
Current hole = dark filled
Future holes = gray outline
Clicking a dot jumps to that hole

3. Hole Illustration

SVG diagram showing fairway shape, bunkers, green
Hole number badge in corner
If no diagram available, show placeholder

4. Hole Info

Hole number (large, serif)
Par, yardage, handicap rating

5. Caddie Notes

Gold left border accent
Serif italic text
1-2 sentences of playing advice
Pull from course database

6. Score Input

List all players in the foursome
Each row: Avatar, Name, Running total, +/- buttons, Score
Score highlighting:

Eagle or better: green background, "Eagle" label
Birdie: green background, "Birdie" label
Par: no highlight
Bogey: yellow background, "Bogey" label
Double+: red background, "Double+" label


Scores auto-save on change

7. Navigation

"← Prev Hole" and "Next Hole →" buttons
"End Round" button (red outline, confirms before ending)

Behavior

When round starts, Caddie Mode activates automatically
Scores save to database on every change
Next Hole saves current hole and advances
End Round confirms, saves final scores, exits to normal LIVE rotation
After round ends, show summary card, then return to news/trip rotation

Data Required
typescriptinterface Round {
  id: string;
  tripId: string;
  courseId: string;
  status: 'not_started' | 'in_progress' | 'complete';
  currentHole: number;
  players: string[]; // user IDs
  scores: {
    odlayerId: string;
    odlayerName: string;
    holeScores: (number | null)[]; // index 0 = hole 1
  }[];
  startedAt: Date;
  completedAt?: Date;
}

interface CourseHole {
  number: number;
  par: number;
  yardage: number;
  handicap: number;
  notes?: string;
  illustration?: string; // SVG path or URL
}
When NOT in a round
LIVE shows the normal card rotation (weather, PGA tour, news, trip info)
