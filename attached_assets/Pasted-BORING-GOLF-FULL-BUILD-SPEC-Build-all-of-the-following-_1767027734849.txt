BORING GOLF — FULL BUILD SPEC
Build all of the following features. Work through them systematically. Don't stop until complete.

1. LIVE SCHEDULE PAGE (/live-schedule)
A gorgeous, single-screen schedule view. Masters broadcast meets PGA Tour app.
Design:

Full-bleed dark background (Fairway #1A2F23)
Subtle topographic pattern at 3% opacity
Trip name in Cormorant Garamond, 28px, centered
Day progress bar (thin gold line with dot showing current position)

Structure:

"NOW" section with elevated current event card (white card, gold left border, subtle glow)
"NEXT" section with upcoming events as a clean list
"TOMORROW" section with future events
Times in JetBrains Mono (brass), event names in Inter (white), locations in muted gray

Current event card features:

Pulsing green "ON COURSE" badge if it's a round
Hole progress (e.g., "Hole 7 of 18")
Course tip in italic (e.g., "Favor the left. Right bunker is a round-killer.")
Player avatar dots at bottom showing who's on course

Use hardcoded Pinehurst data for now.

2. CHARLIE SCORECARD (/charliescorecard)
Premium golf scorecard with live skins tracking.
Design:

Dark theme (Fairway background)
Leaderboard at top showing all players ranked by score to par
Toggle between SCORECARD and SKINS views
Live feed notifications sliding in from bottom

Leaderboard row:

Position number (brass if top 3)
Player avatar circle with initials
Player name
Score to par (green if under, red if over, white if even)
Skins count

Scorecard view:

Horizontal scrolling grid
Hole numbers across top, par below
Player rows with tappable score cells
Color coding: Eagle (gold), Birdie (green), Par (gray), Bogey (light red), Double+ (red)
Running totals on right side

Skins view:

List of all holes played
Shows winner or "Carryover" for each hole
Value of each skin
Bottom bar: "$40 carryover • 7 skins won • $100 pot left"

Skins logic (auto-calculated):

Lowest score on hole wins the skin
Ties = carryover to next hole
$20 per skin default
Notifications when someone wins: "Marcus takes skin on #7 — $20"

Pre-populate with 6 players through 9 holes so it looks live.

## DESIGN SYSTEM REMINDER

**Colors:**
- Fairway #1A2F23 (primary dark)
- Limestone #F5F3EF (light background)
- Brass #C9A227 (accent — use sparingly)
- Fog #E8E6E1 (card surfaces)
- Rough #3D5A47 (secondary green)
- Chalk #FFFFFF
- Dusk #6B7280 (muted text)
- Positive #22A55B
- Warning #F59E0B
- Negative #DC2626

**Fonts:**
- Cormorant Garamond — headlines, trip names
- Inter — body, UI
- JetBrains Mono — numbers, times, money

**Principles:**
- Brass used like a good putter: rarely, decisively
- Generous whitespace
- No gradients
- No emojis in UI
- Animations subtle (150-300ms)
- Mobile-first
